# Deploy Button Snippet

Replace `YOUR_USERNAME/YOUR_REPO` with your public GitHub repository.

```markdown
[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/YOUR_USERNAME/YOUR_REPO)
```

For GitLab, use the public GitLab repo URL instead:

```markdown
[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://gitlab.com/YOUR_USERNAME/YOUR_REPO)
```
